# WhozWho Client Technical Context

## Technologies Used
- TypeScript 5.3.3
- Node.js
- Axios 1.6.7
- Jest 29.7.0
- ESLint 8.57.0
- Prettier 3.2.5

## Development Setup
1. Prerequisites:
   - Node.js
   - npm/yarn
   - Git

2. Installation:
   ```bash
   npm install
   ```

3. Development Commands:
   - `npm run build`: Compile TypeScript
   - `npm test`: Run tests
   - `npm run lint`: Check code style
   - `npm run format`: Format code

## Technical Dependencies
1. Runtime:
